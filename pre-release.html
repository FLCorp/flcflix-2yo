Détails sur FLC Flix Pre-Release

FLC-Flix_0.3 (ou FLC Flix Pre-Release) est sorti courant le mois de novembre 2022. Cette version avait été développée dans le seul but de préparer le terrain pour la sortie de FLC Flix v1. Le site avait donc été totalement revisité pour faire en sorte que le site soit le plus "simple" possible avant la sortie de la première vraie version du site.
